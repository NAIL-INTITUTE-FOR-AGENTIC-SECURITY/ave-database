# 🎓 Grant Strategy

## Overview

Research grants provide non-dilutive funding for the NAIL Institute's
core research activities while building academic credibility and
institutional partnerships. Target: 10% of revenue by Year 3.

## Grant Priorities

### Priority 1: AI Safety & Security Research
- Agentic vulnerability discovery and taxonomy
- Autonomous system threat modelling
- Multi-agent security analysis
- Defence mechanism development and validation

### Priority 2: Standards Development
- Vulnerability scoring methodologies
- Certification framework validation
- Industry benchmark creation
- Regulatory compliance tooling

### Priority 3: Open-Source Tooling
- AVE database infrastructure
- Assessment automation tools
- Community platform development
- API and integration development

## Application Process

### Pre-Application (2–4 weeks)
1. Identify matching funder and programme
2. Review eligibility criteria and past awards
3. Contact programme officer (if possible)
4. Identify collaborators / co-investigators
5. Develop preliminary concept note
6. Get internal approval from Executive Director

### Application (4–8 weeks)
1. Draft research plan using `templates/proposal_template.md`
2. Prepare budget using `templates/budget_justification.md`
3. Collect CVs, letters of support, institutional docs
4. Internal review (peer feedback from advisory board)
5. Revise and finalise
6. Submit via funder's portal
7. Log in `tracker.yaml`

### Post-Submission (2–6 months)
1. Monitor for reviewer queries
2. Prepare for interview/presentation (if applicable)
3. Address reviewer feedback
4. Decision notification → update tracker

### Post-Award
1. Sign grant agreement
2. Set up project accounting codes
3. Kick off research plan
4. Submit reports per grant schedule
5. Acknowledge funder in all publications
