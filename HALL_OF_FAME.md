## ⚡ How XP Works

### Base XP by Severity

| Severity | Base XP |
|----------|--------:|
| 🔴 critical | 500 XP |
| 🟠 high | 300 XP |
| 🟡 medium | 150 XP |
| 🟢 low | 75 XP |
| ℹ️ info | 50 XP |

### Status Multiplier

| Status | Multiplier |
|--------|----------:|
| `proven_mitigated` | ×1.2 |
| `proven` | ×1.0 |
| `theoretical` | ×0.5 |
| `in_progress` | ×0.3 |
| `not_proven` | ×0.1 |

| Action | Bonus |
