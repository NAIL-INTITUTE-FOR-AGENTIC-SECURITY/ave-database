## Pre-Event Checklist

### 8 Weeks Before
- [ ] Confirm instructor availability
- [ ] Book venue / set up virtual platform
- [ ] Open registration
- [ ] Begin marketing (email, social, website)

### 4 Weeks Before
- [ ] Review registration numbers (minimum viable: 8)
- [ ] Order printed materials (if in-person)
- [ ] Confirm catering (if in-person)
- [ ] Send pre-reading materials to registered attendees

### 2 Weeks Before
- [ ] Send logistics email to attendees
- [ ] Finalise lab environments
- [ ] Run full tech rehearsal (if virtual)
- [ ] Prepare name badges and materials packs (if in-person)

### 1 Day Before
- [ ] Final lab environment test
- [ ] Confirm all AV equipment works
- [ ] Print sign-in sheets
- [ ] Prepare feedback survey links

## Day-Of Checklist

### Morning
- [ ] Arrive 1 hour early (in-person) / 30 min early (virtual)
- [ ] Set up room / test platform
- [ ] Test all demos and labs
- [ ] Welcome attendees, distribute materials

### Throughout
- [ ] Follow curriculum timing
- [ ] Collect sign-in signatures
- [ ] Monitor lab progress
- [ ] Take photos (with permission) for social media

### End of Day
- [ ] Distribute feedback survey
- [ ] Administer exam (if applicable)
- [ ] Collect all materials
- [ ] Thank attendees, share next steps

## Post-Event (Within 1 Week)
- [ ] Score exams and submit results
- [ ] Issue certificates
- [ ] Send follow-up email with resources
- [ ] Analyse feedback survey results
- [ ] Submit event report to NAIL
- [ ] Post event summary on social media
