# NAIL Institute — Annual AVE Report

## "State of Agentic AI Security"

The Annual AVE Report is the NAIL Institute's flagship publication — a comprehensive
analysis of the agentic AI vulnerability landscape, trends, defences, and predictions.

## Contributing

### Call for Data

The Annual Report relies on community contributions:

- **AVE Card Analysis**: Submit case studies of vulnerability discoveries
- **Defence Reports**: Share effectiveness data on mitigation strategies
- **Industry Surveys**: Participate in the annual practitioner survey
- **Peer Review**: Volunteer to review draft chapters

### How to Contribute

1. **Submit Data**: Open a PR to `annual-report/2025/data/` with your analysis
2. **Write a Section**: Claim a chapter or subsection via GitHub Issues
3. **Peer Review**: Sign up in `#annual-report` Slack channel
4. **Sponsor**: Contact report@nailinstitute.org for sponsorship opportunities

## Methodology

See [12-methodology.md](2025/12-methodology.md) for complete methodology.

### Data Sources
- NAIL AVE Database (all published cards)
- Community survey (annual, December)
- Academic publication analysis
- Industry incident reports (anonymised)
- Expert panel interviews

### Analysis Tools
- Statistical analysis: Python (pandas, scipy)
- Visualisation: Matplotlib, Plotly
- Report generation: Markdown → PDF pipeline
