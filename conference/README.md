# 🎤 NAIL Institute — Conference Presence Kit

> Everything needed to represent the NAIL Institute at major security
> and AI conferences.

## Overview

This kit provides ready-to-use materials for conference presentations,
booth presence, CTF demonstrations, and networking events at venues
including DEF CON AI Village, Black Hat, NeurIPS, USENIX Security,
IEEE S&P, and regional events.
