## 8 Weeks Before

- [ ] Confirm talk/booth acceptance
- [ ] Book flights (check visa requirements)
- [ ] Book hotel (≤ $300/night, near venue)
- [ ] Register team for conference badges
- [ ] Submit final talk abstract (if required)
- [ ] Order swag (see quantities below)
- [ ] Order print materials (one-pagers, business cards)
- [ ] Create event-specific landing page

## 4 Weeks Before

- [ ] Finalise slide deck (see brand guidelines)
- [ ] Complete live demo environment setup
- [ ] Test demo on conference WiFi equivalent
- [ ] Ship booth materials (if international)
- [ ] Arrange AV requirements with venue
- [ ] Draft social media content calendar
- [ ] Prepare press release (if newsworthy)
- [ ] Brief team on roles and schedules

## 2 Weeks Before

- [ ] Final slide deck review + backup PDF
- [ ] Practice talk (time to ≤ allocated slot minus 5 min)
- [ ] Practice live demo (3 complete run-throughs)
- [ ] Print emergency contact cards for team
- [ ] Confirm all logistics (transfer, hotel, venue)
- [ ] Test all equipment (laptop, adapters, clicker)
- [ ] Pre-write key tweets/posts for during event
- [ ] Prepare conversation starters for booth staff

## Day Before Event

- [ ] Arrive at venue, scout room/booth location
- [ ] Set up booth (if applicable)
  - [ ] Backdrop banner
  - [ ] Pull-up banner
  - [ ] Table cloth + table banner
  - [ ] Monitor/tablet with demo
  - [ ] Swag display
  - [ ] Business card bowl
  - [ ] QR code stand
  - [ ] Power strips + extensions
- [ ] Test demo on venue WiFi
- [ ] Test projector/display connection
- [ ] Charge all devices to 100%
- [ ] Review schedule for next day

## Event Day (Talk)

- [ ] Arrive 30 min before talk
- [ ] Connect laptop, test display
- [ ] Check microphone
- [ ] Load backup PDF on USB drive
- [ ] Verify demo environment is running
- [ ] Have water at podium
- [ ] Start recording (if self-recording)
- [ ] Tweet/post "Going live in 10 min" with room number

### During Talk
- [ ] Start with hook / demo teaser
- [ ] Stick to time (watch timer)
- [ ] End with clear CTA + QR code slide
- [ ] Leave 5 min for Q&A

### After Talk
- [ ] Thank attendees
- [ ] Share slides link (tweet + slide at end)
- [ ] Collect business cards / contact info
- [ ] Post highlights on social media
- [ ] Note questions for FAQ/follow-up

## Event Day (Booth)

- [ ] Open booth 15 min before hall opens
- [ ] Ensure demo is running + looks good
- [ ] Staff rotation schedule (2-hour shifts)
- [ ] Log all lead contacts (name, org, interest)
- [ ] Replenish swag throughout day
- [ ] Take photos for social media
- [ ] Close booth: secure equipment, take remaining swag

## Post-Event (Within 1 Week)

- [ ] Send follow-up emails to leads (within 48 hours)
- [ ] Post event recap on blog/LinkedIn
- [ ] Share talk recording link (when available)
- [ ] Upload slides to SlideShare / GitHub
- [ ] Debrief: what worked, what to improve
- [ ] Update lead tracker / CRM
- [ ] Process expense receipts
- [ ] Write internal trip report
- [ ] Update conference/reports/ with summary

## Equipment Packing List

### Always Bring
- [ ] Laptop (fully charged, backup charger)
- [ ] USB-C to HDMI adapter
- [ ] USB-C to USB-A adapter
- [ ] Presentation clicker
- [ ] USB drive with backup slides (PDF)
- [ ] Extension cord / power strip
- [ ] Phone charger + battery pack
- [ ] Business cards
- [ ] Printed schedule + booth map

### Booth Events
- [ ] All "Always Bring" items
- [ ] External monitor or tablet (for demos)
- [ ] Backdrop banner (in travel tube)
- [ ] Pull-up banner (in carry case)
- [ ] Table cloth (navy, branded)
- [ ] QR code stand / acrylic holder
- [ ] Swag boxes
- [ ] Cable ties / velcro for cable management
- [ ] Gaffer tape
- [ ] Marker pen + paper (for quick notes)
