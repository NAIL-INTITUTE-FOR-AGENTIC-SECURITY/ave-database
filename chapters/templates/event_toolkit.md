# Event Toolkit

Templates and guides for organising NAIL Institute chapter events.

---

## Event Types

### 1. Monthly Meetup (2 hours)

**Format**: Presentation + Discussion + Networking

| Time | Activity |
|------|----------|
| 0:00-0:10 | Welcome & announcements |
| 0:10-0:40 | Featured talk (30 min) |
| 0:40-0:50 | Q&A |
| 0:50-1:00 | Break |
| 1:00-1:30 | Lightning talks (3×10 min) or group discussion |
| 1:30-2:00 | Networking / open floor |

**Checklist**:
- [ ] Book venue or set up virtual meeting link
- [ ] Confirm speaker(s) 2 weeks in advance
- [ ] Post event on NAIL community calendar
- [ ] Send reminder 48 hours before
- [ ] Record session (with speaker consent)
- [ ] Share slides and recording within 1 week
- [ ] Collect feedback (use standard form)

### 2. Workshop / Deep-Dive (Half-day)

**Format**: Hands-on, instructor-led

| Time | Activity |
|------|----------|
| 0:00-0:15 | Introduction & setup |
| 0:15-1:15 | Module 1: Theory & concepts |
| 1:15-1:30 | Break |
| 1:30-2:30 | Module 2: Hands-on exercise |
| 2:30-2:45 | Break |
| 2:45-3:45 | Module 3: Advanced exercise / CTF |
| 3:45-4:00 | Wrap-up & feedback |

**Workshop Topics** (suggested):
- AVE card writing: from discovery to publication
- Red-teaming multi-agent systems
- AVSS scoring methodology
- Building agentic AI guardrails
- Prompt injection defence patterns
- Tool-use security in LLM agents

### 3. Capture-The-Flag (CTF) Event

**Format**: Team-based competition

**Categories**:
- Prompt injection challenges
- Multi-agent exploitation scenarios
- Tool-use abuse detection
- Guardrail bypass challenges
- AVE card identification from scenarios

**Logistics**:
- [ ] Set up CTF platform (CTFd recommended)
- [ ] Prepare 15-20 challenges across 3 difficulty levels
- [ ] Recruit 2-3 challenge reviewers
- [ ] Arrange prizes (NAIL swag, conference tickets)
- [ ] Plan live scoreboard display
- [ ] Prepare post-event writeups

### 4. Annual Regional Conference (Full-day)

See the [Conference Programme](../../conference/) for the global template.
Adapt tracks and timing for regional audience.

## Templates

### Event Announcement Template

```markdown
# [Event Name] — NAIL [Region] Chapter

📅 **Date**: [Date & Time with timezone]
📍 **Location**: [Venue / Virtual link]
🎫 **Registration**: [Link]

## About

[2-3 sentence description of the event]

## Agenda

[Copy from appropriate event type above]

## Speaker(s)

**[Speaker Name]** — [Title, Organisation]
[1-2 sentence bio]

## Who Should Attend

- AI/ML engineers building agentic systems
- Security researchers focused on AI threats
- Anyone interested in agentic AI safety

## Registration

[Free / Paid] — [Registration link]

## Code of Conduct

All NAIL events follow the [NAIL Institute Code of Conduct](../../CODE_OF_CONDUCT.md).
```

### Speaker Invitation Template

```markdown
Subject: Speaking Invitation — NAIL [Region] [Event Type]

Dear [Name],

I'm writing on behalf of the NAIL Institute [Region] Chapter to invite you
to present at our upcoming [event type] on [date].

We'd love for you to share your expertise on [topic]. The session would be
[duration] with [format: presentation/workshop/panel].

**Event Details:**
- Date: [Date]
- Expected audience: [N] attendees
- Format: [Virtual/In-person/Hybrid]
- Recording: [Yes/No, with permission]

We can offer [travel support / honourarium / NAIL membership / swag].

Please let us know if you're interested, and we'll share more details.

Best regards,
[Your Name]
NAIL Institute [Region] Chapter Lead
```

### Feedback Form Questions

1. Overall, how would you rate this event? (1-5 stars)
2. How relevant was the content to your work? (1-5 stars)
3. What was the most valuable part of the event?
4. What could be improved?
5. What topics would you like to see at future events?
6. Would you recommend NAIL events to a colleague? (Yes/No)
7. Any other comments?

## Branding

- Use the NAIL Institute logo with regional chapter designation
- Slide deck template: request from chapters@nailinstitute.org
- Colours: NAIL blue (#1a365d), accent teal (#0d9488), white (#ffffff)
- Font: Inter (headings), system sans-serif (body)

## Budget Guidelines

| Item | Typical Cost | Notes |
|------|-------------|-------|
| Venue hire | $0-500 | Seek sponsors or university hosts |
| Virtual platform | $0-50/mo | Zoom, Google Meet, or Discord |
| Catering (in-person) | $10-20/person | Light refreshments |
| Speaker travel | $0-1000 | For out-of-region guests |
| Swag / prizes | $100-500 | NAIL stickers, t-shirts |
| Recording / streaming | $0-200 | OBS is free for virtual |
