# NAIL Institute — Brand Guidelines for Regional Chapters

## Chapter Naming

### Official Name Format
```
NAIL Institute — [Region] Chapter
```

### Examples
- NAIL Institute — North America Chapter
- NAIL Institute — Europe Chapter
- NAIL Institute — Asia-Pacific Chapter

### Abbreviations
- NAIL NA, NAIL EU, NAIL APAC (for informal use only)

### What NOT to Do
- ❌ "NAIL North America" (missing "Institute" and "Chapter")
- ❌ "North American NAIL" (wrong order)
- ❌ "NAIL Inc. NA Chapter" (NAIL is not "Inc.")

## Logo Usage

### Primary Logo
- Use the NAIL Institute logo with the regional chapter lockup
- Regional designation appears below the primary logo in regular weight
- Request logo files from chapters@nailinstitute.org

### Colour Palette

| Colour | Hex | Usage |
|--------|-----|-------|
| NAIL Blue | `#1a365d` | Primary brand colour, headings |
| NAIL Teal | `#0d9488` | Accent, links, CTAs |
| Dark Grey | `#1a202c` | Body text |
| Light Grey | `#f7fafc` | Backgrounds |
| White | `#ffffff` | Cards, content areas |
| Danger Red | `#e53e3e` | Alerts, critical severity |
| Warning Amber | `#d69e2e` | Warnings, high severity |

### Typography
- **Headings**: Inter Bold / Semi-Bold
- **Body**: Inter Regular / system sans-serif
- **Code**: JetBrains Mono / system monospace
- **Minimum size**: 14px body, 12px captions

### Clear Space
- Maintain minimum clear space of 1× logo height around the logo
- Never place the logo on busy backgrounds without a container

## Slide Deck Template

Request the official NAIL slide deck template from chapters@nailinstitute.org.
Includes:
- Title slide with chapter name
- Content slides (text, image, split layout)
- Code slides with syntax highlighting
- AVE card showcase slide
- Speaker bio slide
- Closing / Q&A slide

## Social Media

### Handles
- Chapters should not create separate social media accounts
- Use the main NAIL Institute accounts with chapter-specific hashtags
- Hashtags: `#NAILInstitute`, `#NAILChapter[Region]`, `#AgenticSecurity`

### Posting Guidelines
- Always tag @NAILInstitute in chapter-related posts
- Use official imagery and colour palette
- Link back to nailinstitute.org for all resources
- Do not make official statements on behalf of NAIL without approval

## Email Signatures

```
[Your Name]
[Role], NAIL Institute — [Region] Chapter
[email]@nailinstitute.org
https://nailinstitute.org/chapters/[region]
```

## Co-Branding

When co-branding with partner organisations:
- NAIL logo must appear at equal or greater prominence
- Use "In partnership with" or "Supported by" language
- Both logos must maintain their respective clear spaces
- Submit co-branded materials to chapters@nailinstitute.org for review

## Merchandise

- All NAIL merchandise must be approved by the Global Chapters Committee
- Use the official NAIL print files (request from brand@nailinstitute.org)
- Approved items: stickers, t-shirts, tote bags, notebooks
- Regional chapters may add their region name to approved designs
