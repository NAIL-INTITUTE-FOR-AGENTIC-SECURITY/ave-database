# AVE-2025-0004: Prompt Inbreeding

| Field | Value |
|-------|-------|
| **Severity** | 🟡 medium |
| **Category** | `drift` |
| **Status** | `proven_mitigated` |
| **Discovered** | 2025-01 |

## Summary

Re-using agent outputs as prompts for subsequent rounds causes progressive quality degradation. Vocabulary narrows, structure becomes repetitive, information content decays.

## Blast Radius

Complete loss of output diversity and information value.

## Prerequisites

Multi-round system where outputs feed back as inputs.

## Mechanism

Each generation of output becomes the input for the next. [Available in NAIL SDK]
