## Summary

Model upgrades introduce silent behavioural regressions. Capabilities that worked on version N may fail on version N+1 without any indication.

## Blast Radius

Silent production failures after routine model upgrades.

## Prerequisites

Model version change in production.

New model versions change response distributions. [Available in NAIL SDK]
