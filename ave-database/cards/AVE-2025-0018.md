## Summary

Agents have no awareness of their infrastructure state. When GPU throttles, RAM fills, or network times out, the agent misattributes the failure as a code bug and hallucinates fixes.

## Blast Radius

Cascading incorrect diagnoses. Wasted effort on phantom bugs.

## Prerequisites

Agent without infrastructure telemetry in context.

## Mechanism

The agent's context contains no hardware telemetry. [Available in NAIL SDK]
